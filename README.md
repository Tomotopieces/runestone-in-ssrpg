### 双词缀
指装备的词缀同时拥有`符文词缀`和`通用词缀`
> 例子1：dP毒属性大锤，对体力属性敌人有小幅度的伤害增加，且攻击命中目标（目标的属性不限）后，在一定时间内降低其攻击力。

> 例子2：A以太属性盾，每与一名新出的火属性敌人战斗时，增加一些自身的铠甲

注：A或者D词缀只能单独存在
